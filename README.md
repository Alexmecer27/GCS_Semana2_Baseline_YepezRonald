# Proyecto GCS - Baseline v1.0 - YepezRonald

## Objetivo
Implementar un repositorio con buenas prácticas de gestión de configuración, estableciendo una línea base auditable y trazable para la materia de Gestión de Configuración de Software.

## Estructura
- **/docs**: Documentación de requisitos (SRS) y diseño (SDD).
- **/src**: Código fuente del proyecto.
- **/config**: Plantillas de configuración controlada.
- **/tests**: Pruebas unitarias (placeholders).

## Cómo crear la Baseline
Para generar la línea base aprobada:
`git tag -a v1.0 -m "Baseline v1.0: SRS+SDD approved + minimal build"`
