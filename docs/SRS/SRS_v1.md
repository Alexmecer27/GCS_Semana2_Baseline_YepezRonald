# Especificación de Requisitos de Software (v1.0)

## Requisitos Funcionales
1. **REQ-001**: El sistema debe validar el formato de correo electrónico en el login.
2. **REQ-002**: El usuario debe poder ver una lista de productos disponibles.
3. **REQ-003**: El sistema debe permitir agregar ítems al carrito de compras.
4. **REQ-004**: El sistema debe generar un total de compra antes de pagar.

## Requisitos No Funcionales
1. **RNF-001**: La interfaz debe cargar en menos de 2 segundos en redes 4G.
2. **RNF-002**: El código debe estar documentado siguiendo el estándar JSDoc/DocString.
5. REQ-007: Exportar reportes a PDF.
